##  Introduction

A sleek and secure password generator built with Python and web technologies. This tool helps users create strong, customizable passwords instantly. Designed with simplicity and usability in mind, it’s perfect for personal use or integration into larger systems.

---

##  Summary

This project showcases how Python can power a lightweight yet effective password generation tool with a clean web interface. It allows users to:

- Generate passwords of varying lengths
- Choose character sets (uppercase, lowercase, digits, symbols)
- Copy passwords with a single click
- Enjoy a responsive, mobile-friendly design

---

##  Project Overview

The password generator is structured to highlight:

- **Security-first logic**: Randomized password creation using Python's `random` module
- **Interactive UI**: HTML/CSS frontend with JavaScript enhancements
- **Backend simplicity**: Flask-powered backend for deployment flexibility
- **Deployment**: Hosted on Netlify for fast and free access

---

##  Features

-  Customizable password length
-  Toggle character types (A-Z, a-z, 0-9, symbols)
-  Copy-to-clipboard functionality
-  Responsive design for all devices
-  Fast online access via Netlify

##  Getting Started

### Clone the Repository

```bash
git clone https://github.com/souraaav/passwordgenarator.git
cd passwordgenarator
```

### Run Locally

```bash
python app.py
```

Then open `http://localhost:5000` in your browser.
